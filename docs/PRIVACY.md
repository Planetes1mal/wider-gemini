# Privacy Policy for Wider Gemini

Last updated: 2025.11.12

Wider Gemini ("we", "us", or "our") allows users to customize the interface of Google Gemini.

**Information Collection and Use**
We do not collect, store, or transmit any personal information, chat data, or browsing history. All settings (width preferences) are stored locally on your device using Chrome's Sync Storage API and are only used to maintain your preferences across your synchronized browser instances.

**Third-Party Services**
This extension interacts with the Google Gemini website strictly to modify its CSS/Styling for display purposes. No data is sent to external servers.

**Contact**
If you have any questions about this Privacy Policy, please contact us via our GitHub repository.
