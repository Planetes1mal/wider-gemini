🔍 Gemini 代码区域样式提取器

专门用于分析 Gemini 回答中的代码块样式信息

📋 本工具将提取以下信息：

  • ✅ 代码块容器的样式（pre、code 标签）
  • ✅ 代码内容区域的样式（white-space、word-wrap 等）
  • ✅ 复制按钮的位置和样式
  • ✅ 代码块的父容器结构
  • ✅ 所有相关的 CSS 类名和属性
  • ✅ 滚动条和溢出处理方式
⚠️ 重要提示：
请确保在 Gemini 页面中有包含代码块的回答，否则可能提取不到数据！
建议先向 Gemini 提问一个编程问题，等待它生成包含代码的回答后再运行脚本。

📖 使用步骤：

  1. 打开 Gemini 页面（https://gemini.google.com/app）
  2. 确保页面中有包含代码块的回答（如果没有，先问一个编程问题）
  3. 按 F12 打开浏览器开发者工具
  4. 切换到 Console（控制台） 标签
  5. 点击下方"生成提取脚本"按钮
  6. 点击"复制到剪贴板"按钮
  7. 在 Gemini 页面的控制台中 粘贴并按回车 运行
  8. 脚本会自动下载一个 JSON 文件
  9. 将下载的 JSON 文件发送给我进行分析
